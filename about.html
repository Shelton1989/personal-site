About this site

This site was written in HTML 5 and uses a CSS framework called Materialize. It is a lesser known framework, most developers use the bootstrap framework but Materialize was developed by Google. The style of this site could've been written from scratch but I chose to use a framework because of the following reasons:

  • - They are made so that you don't have to rewrite code.
  • - They are always maintained by the creators and therefore easy to maintain.
  • - They are modern and easy to incorporate.

This Site includes the JQuery library but I haven't incorporated it because all the Javascript on this site is written in pure Javascript. Mainly because it's a force of habit. The main components using JavaScript:

  • - The home page slide.
  • - The contact page form
  • - The services page cards

The slide is pretty much basic JS. It's essentially just a counter variable adding on numbers to an image id so that the large image inherits the src attribute value of the the thumbnail. There in also an interval set so that the slide show runs on its own.

The contact page is fairly simple. It uses emailJS methods for sending mails from an API. Basically, The form is a standard HTML form. It collects all the information from the user, a JS variable stores it in a JSON object and emailJS puts it in a template and sends it to a recipient email. Easy.

The cards on the service page were not written by me but rather by Materialize. I assume it works the same way that the Bootstrap collapsible menu does.
